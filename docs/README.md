# 📚 Documentación - Artificial Intelligence Society (AIS)

Bienvenido a la carpeta de documentación del sitio web oficial de **AIS (Artificial Intelligence Society)**. Este espacio está diseñado para registrar la arquitectura, guías de diseño, decisiones técnicas y guías del desarrollador para el mantenimiento del sitio.

---

## 📁 Estructura de Documentación Recomendada

Puedes organizar y encontrar información en las siguientes secciones (puedes crear nuevos archivos a medida que crezca el proyecto):

* [**Arquitectura y Tecnologías**](./architecture.md): Detalle técnico del stack utilizado (Astro, Tailwind CSS, Vercel).
* **Guías de Estilo y UI**: Paleta de colores HSL, tipografía, micro-animaciones y diseño neón.
* **Guías de Despliegue**: Instrucciones para compilar y desplegar en Vercel.

---

## 🛠️ Stack Tecnológico

El proyecto está construido bajo los siguientes estándares de desarrollo web moderno:

1. **Astro (v4+)**: Framework web orientado al rendimiento que utiliza arquitectura de islas para entregar HTML estático rápido.
2. **Tailwind CSS**: Framework CSS utilitario para un estilado ágil, responsivo y coherente.
3. **Vercel Adapter**: Configurado para renderizado de servidor híbrido y despliegues serverless fluidos.

---

## 🚀 Comandos Rápidos del Proyecto

En la raíz del proyecto, puedes utilizar los siguientes comandos de terminal:

| Comando | Acción |
| :--- | :--- |
| `npm install` | Instala todas las dependencias del proyecto. |
| `npm run dev` | Inicia el servidor de desarrollo local en `http://localhost:4321`. |
| `npm run build` | Compila el sitio optimizado para producción en la carpeta `.vercel/`. |
| `npm run preview` | Previsualiza localmente la compilación de producción. |
